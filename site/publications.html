1. RGNN3D: A hybrid radiomic graph neural network for 3D MRI glioma grading

Published in Knowledge-Based Systems (Elsevier, Q1, IF: 7.6, First Author), 2026

DOI: 10.1016/j.knosys.2026.115343

Description: This study proposes RGNN3D, a deep neural network that integrates Graph Neural Networks and LSTM to accurately grade gliomas using 3D MRI radiomic biomarkers. By extracting 112 radiomic features and leveraging graph-based learning, the model achieves 98.58% accuracy. An explainable AI method (LIME) is also applied to identify key biomarkers, supporting radiologists in more reliable glioma diagnosis.

2. AMDNet23: Hybrid CNN-LSTM deep learning approach with enhanced preprocessing for age-related macular degeneration (AMD) detection

Published in Intelligent Systems with Applications (Elsevier, Q1, IF: 4.3, First Author), 2024

DOI: 10.1016/j.iswa.2024.200334

Description: This study introduces AMDNet23, a hybrid CNN–LSTM framework for automated detection of Age-related Macular Degeneration (AMD) from fundus images. Using CLAHE and Gamma correction for image enhancement, the model was evaluated on 2000 images across four classes and achieved 96.50% accuracy, outperforming several pre-trained CNN models and demonstrating strong potential for automated ocular disease diagnosis.
